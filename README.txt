This package simulates a turtlebot which can response to the postion of a detected object( a blue ball. etc)
Nodes it will launch:
	uvc_camera ( get video stream )
	coordinate_transform (object detection and publish transformed coordinate to topic /obj_pos

Test Ball:
	color: dark green;
	radius: 3.5cm
	H: 70-112
	S: 26 - 171
	V: 229 - 86
	HC_param1: 200
	HC_param2: 33
